Cadence is a lightweight agent harness for everyday use. It helps you run and manage an agent on Windows with a simple setup and a stable runtime.

## 💻 System requirements

Cadence is built for Windows and works best on a modern PC.

You will need:

- Windows 10 or Windows 11
- A recent Intel or AMD processor
- 4 GB of RAM or more
- At least 200 MB of free disk space
- Internet access for the first download

For smooth use, keep Windows updated and close unused apps before you start Cadence.

## 🪟 Install on Windows

If you downloaded a `.zip` file:

1. Open File Explorer
2. Go to your Downloads folder
3. Right-click the `.zip` file
4. Select Extract All
5. Choose a folder and extract the files
6. Open the extracted folder
7. Double-click the cadence app file to start it

If you downloaded a `.exe` file:

1. Go to your Downloads folder
2. Double-click the `.exe` file
3. If Windows asks for permission, select Yes
4. Wait for the app to open

If Windows shows a SmartScreen prompt, choose More info, then Run anyway if you trust the file source.

## 🔧 Basic troubleshooting

If cadence does not open:

1. Make sure the download finished
2. Check that you extracted all files if you used a `.zip`
3. Try running the app again
4. Restart your PC and try once more

If Windows blocks the app:

1. Right-click the app file
2. Select Properties
3. If you see an Unblock option, turn it on
4. Click Apply and try again

If the app opens but does not run as expected:

1. Close cadence
2. Open it again
3. Check your internet connection
4. Look for a log file in the logs folder
5. Try a fresh download from the releases page

If you still have trouble, delete the app folder, download the newest release, and set it up again

## 🔐 Safety and file use

Keep the app files in one folder so cadence can find its settings and logs. Do not move files around after setup unless you need to reinstall.

If you use an antivirus tool, it may scan the app when you first open it. That is common for downloaded files. Let the scan finish before you start the app.
